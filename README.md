# 單字訓練機 Vocabulary Trainer

## 簡介 Introduction
單字就如同蓋房子的磚頭數量一樣，是英文能力的一種根基，單字知道得越多，解析某些文章或許會更加流暢與快速，此程式透過隨機產生單字來達到學習，遇到沒學過或是想背誦的單字可以新增到單字清單中，只要想要複習單字即可點開程式進行學習或複習。
